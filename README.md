# calculatorr
for using you need python3 up to date                                                                                   
you will also need some dependencies                                                                                    
to install it just do :                                                                                                 
python3 -m venv venv                                                                                                    
source venv/bin/activate # on Linux/macOS                                                                              
venv\Scripts\activate # on Windows                                                                                      
pip install -r requirements.txt                                                                                         
deactivate
